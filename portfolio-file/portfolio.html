Application Of Graphic Design



Adobe Illustrator


What it feels like:


"Illustrator is my go-to for creating graphics that can be scaled up or down without losing quality. It's where I make logos, icons, and illustrations."

Tools/Functions:


Selection Tools: "I use the regular selection tool to select entire objects, and the direct selection tool to edit individual points and paths. This is how I fine-tune shapes."
Pen Tool: "The pen tool is tricky, but it's powerful. I use it to draw precise shapes and curves. It's essential for creating vector graphics."
Shape Tools: "These are useful for creating basic shapes quickly. I use them as building blocks for more complex designs."
Type Tool: "I use this for adding text to my illustrations and logos. Illustrator has good options for manipulating text."
Pathfinder: "The Pathfinder helps me combine shapes, like merging them or subtracting them. It's really useful for creating complex forms."

How i use it:


"I use Illustrator to design logos, create illustrations, and make graphics for websites and print. It's where I create things that need to be scalable."

Adobe Photoshop


What it feels like:


"Photoshop is my tool for working with images. I use it for editing photos, creating digital art, and manipulating images."

Tools/Functions:


Selection Tools: "I use these to select parts of an image, like if I want to edit a specific area. There are different selection tools for different situations."
Brush Tool: "The brush tool is for painting and drawing. I can customize the brush to get different effects."
Layers: "Layers are really important. They allow me to work on different parts of an image without affecting others. It's how I keep my work organized."
Image Adjustments: "I use these to change the color, brightness, and contrast of photos. It's how I make photos look better."
Type Tool: "I use this to add text to images. Photoshop has good options for styling text."

How i use it:


"I use Photoshop to edit photos, create graphics for the web, and make mockups. It's where I work with raster images and pixel-based art."



Adobe InDesign


What it feels like:


"InDesign is where I bring everything together. It's like my digital layout table. I use it when I have a lot of text and images that need to be arranged precisely."

Tools/Functions:


Type Tool: "I use this all the time for creating paragraphs, headings, and making sure the text looks good. It's how I control the font, size, and spacing."
Frame Tools: "These are essential for placing images and text. I use the rectangle frame mostly, but the others are useful too. They help me define where things go."
Selection Tool: "This is how I move things around, select them, and resize them. It's my basic tool for interacting with the layout."
Pages Panel/Master Pages: "I've used master pages to create templates for things like page numbers and headers. It makes it easier to keep the design consistent throughout a document."
Image Placement: "Importing images is straightforward. InDesign is good at handling high-resolution images for printing."

How i use it:


"I use InDesign for creating brochures, posters with a lot of text, and multi-page documents. It's where I finalize the design and prepare it for printing."
